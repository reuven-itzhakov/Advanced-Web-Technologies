How to Use Tx2UML

Overview

Tx2UML is a tool designed to help you create class diagrams from text input. You can write your class definitions in a simple syntax, and the tool will generate a visual representation of your classes and their relationships.

Features

  • Toggle between light and dark themes for better visibility.
  • Write class definitions in the text area and see the diagram generated on the canvas.
  • Save, edit, and export your diagrams.
  • Access syntax help and examples to get started quickly.

Syntax Guide

Use the following syntax to define your classes:

  • Class Definition: a ClassName is a class
  • Attributes: ClassName has attribute1, attribute2, ..., attributeN
  • Methods: ClassName can method1, method2, ..., methodN
  • Method Parameters: when ClassName does method pass param1, param2, ..., paramN
  • Inheritance: a ClassName is a ParentClass
  • Relationships: one ClassA is related to many ClassB (1-to-many), one ClassA is related to one ClassB (1-to-1)
  • Connection Class: for each ClassA and ClassB there is a ClassC

Example:

a Person is a class.
Person has name, age, height, weight.
Person can jump, speak, run.
when Person does jump pass howFar, howHigh.
when Person does speak pass sentence.

a Student is a Person.
Student has major, yearStarted.
Student can study.
when Student does study pass subject.

a Lecturer is a Person.
Lecturer can teach.

a Course is a class.
a Grade is a class.
Grade has grade, dateGiven.
one Course is related to many Student.
for each Course and Student there is a Grade.
one Course is related to one Lecturer.

Steps to Create a Diagram

  1. Go to the diagram editor.
  2. Write your class definitions in the text area on the right.
  3. Click the "Edit" or "Save" button to update or save your diagram.
  4. Use the "Export" button to download your diagram as an image.
